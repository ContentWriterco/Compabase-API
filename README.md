# Compabase OpenAPI Specification

Official OpenAPI specification for the Compabase API (`v1`).

## Files

- `openapi.yaml` — source of truth for the API contract.

## Base URL

`https://compabase.com/api/v1/`

## Interactive API Docs

- [Compabase Docs](https://compabase.com/docs/)

## Authentication

API key is required for most endpoints.

Supported headers:

- `X-API-Key: <your_key>`
- `Authorization: Bearer <your_key>`

To request API access, use the contact form:
- [https://compabase.com/api](https://compabase.com/api)

## Notes

- This repository contains only the public API specification.
- The rendered docs UI and backend implementation live in the main Compabase application repository.
